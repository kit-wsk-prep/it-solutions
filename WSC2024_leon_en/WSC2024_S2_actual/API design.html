Belle Croissant Lyonnais - Customer Management API

Endpoint: /api/customers

Methods

  • GET: Retrieve customer information
  • POST: Create a new customer
  • PUT: Update an existing customer
  • DELETE: Delete a customer

Request Parameters

GET

Parameter Type Description
id integer (optional) If provided, returns details of a specific customer

POST/PUT

Parameter Type Description
FirstName string (required, max 50 characters) Customer's first name
LastName string (required, max 50 characters) Customer's last name
Email string (required, max 100 characters, must be unique) Customer's email address
Phone string (optional, max 20 characters) Customer's phone number
LoyaltyPoints integer (optional, non-negative) Customer's loyalty points
RegistrationDate date (required, YYYY-MM-DD format) Date when the customer registered

DELETE

Parameter Type Description
id integer (required) The ID of the customer to delete

Error Handling

  • 400 Bad Request: Invalid input parameters or data types
  • 404 Not Found: Customer not found (for GET, PUT, DELETE with specific ID)
  • 409 Conflict: Email already exists (for POST and PUT)
  • 500 Internal Server Error: Unexpected server error
